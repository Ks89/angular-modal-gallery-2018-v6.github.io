PlainGallery

PlainGallery component contains clickable thumbnails to open modal gallery.


PlainImage

Interface that represent the optional plain image used to open modal image, clicking on it.

If you don't provide this object, angular-modal-gallery will scale down the img path of modal image to display the thumbnail.


img
Type: string

Mandatory source path of the plain image.
description
Type: stringoptional

Optional description text. To ignore this feature, don't add this property.
title
Type: stringoptional

Optional title text. It sets 'title' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property.
alt
Type: stringoptional

Optional alt text. It sets 'alt' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property.
arialLabel
Type: stringoptional

Optional aria label text. It sets 'aria-label' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property.
size
Type: Sizeoptional

Optional object to change the size of modal image. 'width: 50px' and 'height: auto' by default.

PlainGalleryConfig

Interface to configure plain gallery.


strategy
Type: PlainGalleryStrategy

Enum that represents the strategy used to show thumbnails. 'ROW' by default.
layout
Type: PlainGalleryLayout

It can be either 'LineLayout', 'GridLayout' or 'AdvancedLayout' to configure the thumbnail's layout.
advanced
Type: AdvancedConfigoptional

Optional object used to configure 'AdvancedLayout'. It will be ignored with both 'LineLayout' and 'GridLayout'.

PlainGalleryLayout

Empty interface implemented by both 'LineLayout', 'GridLayout', 'AdvancedLayout'.


LineLayout

Class to configure a linear plain gallery.


breakConfig
Type: BreakConfig

Object to configure length and wrap.
justify
Type: string

Represent the flexbox 'justify-content' property. For more info check here
size
Type: Size

Object to change the size of thumbnails. 'width: 50px' and 'height: auto' by default.

GridLayout

Class to configure a grid plain gallery.


breakConfig
Type: BreakConfig

Object to configure length and wrap.
size
Type: Size

Object to change the size of thumbnails. 'width: 50px' and 'height: auto' by default.

AdvancedLayout

Class to configure a custom plain gallery.


modalOpenerByIndex
Type: number

Index of the image to open as current image of the modal gallery.
hideDefaultPlainGallery
Type: boolean

If true, default plain gallery is hidden, otherwise not. It's really recommended to use always true.

BreakConfig

Interface to limit the number of items of the plain gallery or to fill other lines.


length
Type: number

Number of items to show. -1 by default.
wrap
Type: boolean

If true, images will wrap, otherwise not. This property correspond to css 'flex-wrap'. false by default.

AdvancedConfig

Interface to use <a> tags instead of <img> and to customize the css background property for thumbnails in plain gallery.


aTags
Type: boolean

Is true, thumbnails will be <a> tags instead of <img>. false by default.
additionalBackground
Type: string

Value useful to change the css background property. '50% 50%/cover' by default. For more info check here HERE

PlainGalleryStrategy

Enum that represent the strategy used to show thumbnails in plain gallery. It can be one of these enum values:


  • ROW: to show thumbnails in a row horizontally
  • COLUMN: to show thumbnails in a row vertically
  • GRID: to show thumbnails in a grid
  • CUSTOM: to show thumbnails with a custom logic
