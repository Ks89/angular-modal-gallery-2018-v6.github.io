Previews

Previews component contains previews with navigation arrows.


PreviewConfig

Interface to configure previews (not 'side-previews', but those at the bottom) in modal gallery.


visible
Type: boolean

If false, previews will be hidden, otherwise not. true by default.
number
Type: numberoptional

Number of previews to show. 3 by default.
arrows
Type: booleanoptional

If false, preview's navigation arrows are visible, otherwise not. false by default.
clickable
Type: booleanoptional

If false, previous won't be clickable, otherwise not. true by default.
size
Type: Sizeoptional

You can change the default size of previews. width: 50px and height: auto by default.
