Getting Started

Dependencies

The most important required dependencies are:

  • Angular (requires Angular version 4 or higher, tested with 5.2.0)
  • Rxjs 5.0.0 or higher

Installation

After installing the above dependencies, add both hammerjs, mousetrap and angular-modal-gallery running this command:

npm i --save angular-modal-gallery hammerjs mousetrap

Finally add also some additional @types running:

npm i --save-dev @types/hammerjs @types/mousetrap

Configuration

Once finished to install dependencies, you need to import the main module and both hammerjs and mousetrap.

        import 'hammerjs';
        import 'mousetrap';
        import { ModalGalleryModule } from 'angular-modal-gallery';

The only remaining part is to list the imported module in your root module and any additional application modules that make use of the components in this library. The exact method will be slightly different for the root (top-level) module for which you should end up with the code similar to (notice ModalGalleryModule.forRoot()):

        import 'hammerjs';
        import 'mousetrap';
        import {ModalGalleryModule} from 'angular-modal-gallery';
        @NgModule({
          imports: [ModalGalleryModule.forRoot(), ...],
          declarations: [ AppComponent, ... ],
          providers: [ ... ],
          bootstrap: [ AppComponent ]
        })
        export class AppModule {}

Other modules in your application should simply import ModalGalleryModule (without .forRoot()):

        import {ModalGalleryModule} from 'angular-modal-gallery';

        @NgModule({
          imports: [ModalGalleryModule, ...],
          declarations: [ AnotherComponent, ... ]
        })
        export class OtherModule {}

Don't use SystemJS in production!

If you are using SystemJS, you should also adjust your configuration to point to the UMD bundle.

In your SystemJS config file, add both map and packages to tell the System loader where to look for libraries:

        map: {
          'hammerjs'                  : 'node_modules/hammerjs',
          'mousetrap'                 : 'node_modules/mousetrap'
          'angular-modal-gallery'     : 'node_modules/angular-modal-gallery/bundles',
        }

        packages = {
          'hammerjs'                  : { main: 'hammer.js' },
          'mousetrap'                 : { main: 'mousetrap.js' },
          'angular-modal-gallery'     : { main: 'angular-modal-gallery.umd.js' }
        }

For more information, you should check the official SystemJS example.
